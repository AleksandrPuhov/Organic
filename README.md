# surface-1

Surface-1 is a Landing page.

## Installation

Clone this repo to your local machine using https://github.com/AleksandrPuhov/surface-1.git

Use the npm package manager.

```npm
$ npm install
```

## Usage

Use start to work.

```npm
$ npm run start
```

Use build to create production version.

```npm
$ npm run build
```
